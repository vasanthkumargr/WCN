Section 10.1

Narrowband MIMO Model

Why multiple antennas? What do they give us? And how do we write the math for a system with many transmitters and receivers?

From SISO to MIMO

Traditional systems have 1 transmit + 1 receive antenna: SISO. MIMO = Multiple Input Multiple Output.

Multiplexing Gain

Send multiple independent data streams simultaneously. Higher data rate, no extra bandwidth needed.

Diversity Gain

If one path fades, others stay strong. Improves reliability. (Chapter 7.)

🚀
Why MIMO created excitement: Winters, Foschini, Foschini & Gans, and Telatar showed spectral efficiency increases remarkably with multiple antennas — the pioneering works that sparked the MIMO revolution.

The Core MIMO Equation

y = Hx + nThe fundamental MIMO system model
Dimensions:
  y  [Mr×1] = received signal vector
  H  [Mr×Mt] = channel gain matrix
  x  [Mt×1] = transmitted signal vector
  n  [Mr×1] = noise vector

Example (Mt=3, Mr=2):
  y₁ = h₁₁x₁ + h₁₂x₂ + h₁₃x₃ + n₁
  y₂ = h₂₁x₁ + h₂₂x₂ + h₂₃x₃ + n₂
  ↑ Every Rx antenna sees a MIX of ALL Tx signals!
hᵢⱼ = channel gain from transmit antenna j to receive antenna i

Noise and Power

Noise: complex Gaussian, mean 0, covariance σ²·IMr (σ² = N₀/2). Normalise σ²=1 so ρ = P/σ² = average SNR per receive antenna.

Tr(Rx) = ρ    where Rx = E[xxᴴ]Power constraint — total transmit power equals ρ · Trace = sum of diagonal = total power

Channel State Information (CSI)

TypeMeaningHow obtained
CSIRReceiver knows HPilot sequences — easy, almost always assumed
CSITTransmitter knows HFeedback from Rx OR channel reciprocity — harder
No CSINeither side knows HUse statistical model

ZMSW Model

When channel is unknown: Zero Mean Spatially White — each hᵢⱼ ~ CN(0,1), i.i.d. This is the Rayleigh fading model. No spatial bias, no correlation.

Circularly symmetric: distribution is unchanged by rotation in complex plane. Implies mean=0, real and imaginary parts i.i.d.

ML Decoding Complexity

Signals mix at the receiver. Optimal detection = Maximum Likelihood: try all possible transmitted combinations. Complexity = |X|^Mt — exponential!

Problem: Mt=4, 16-QAM (|X|=16) → 16⁴ = 65,536 combinations per received vector. Typically prohibitive. If H is known at transmitter, complexity is dramatically reduced (next section).
Practice — §10.1
  1. A 4×3 MIMO: how many Tx and Rx antennas? Maximum rank of H?
  2. Mt=4, QPSK (|X|=4): what is ML complexity? Why is this problematic?
  3. Why is CSIR easier to obtain than CSIT in a real mobile system?

1. H is Mr×Mt=4×3 → Mr=4 Rx, Mt=3 Tx. Max rank = min(3,4)=3.

2. 4⁴=256 combinations per vector. Real-time mobile systems can't evaluate 256 candidates per symbol — much worse with more antennas.

3. Receiver sees y=Hx+n. By sending known pilot symbols (x known), it solves for H directly. Transmitter would need H sent back via feedback (adding delay, error, bandwidth overhead) or require channel reciprocity (TDD only).

Section 10.2

Parallel Decomposition of the MIMO Channel

Convert one messy mixed MIMO channel into several clean independent pipes using SVD. This is where MIMO becomes truly powerful.

The Key Idea — Multiplexing Gain

A MIMO channel can be transformed into RH independent parallel SISO channels. Sending RH independent streams → RH× data rate of SISO.

Multiplexing Gain = R_H = rank of H ≤ min(Mt, Mr)The number of independent parallel channels you can create

Singular Value Decomposition (SVD)

H = U Σ VᴴEq. (10.2) — Every matrix can be decomposed this way
MatrixSizePropertyRole
UMr×MrUᴴU = I (unitary)Receiver rotation
ΣMr×MtDiagonal, σᵢ ≥ 0Channel strengths
VMt×MtVᴴV = I (unitary)Transmitter rotation
Σ = [σ₁  0   0 ]
    [0   σ₂  0 ]    σᵢ = √λᵢ (λᵢ = eigenvalue of HHᴴ)
    [0   0   σ₃]    R_H nonzero singular values
Singular values reveal how strong each parallel channel is

Full rank (rich scattering): RH = min(Mt,Mr) — maximum parallel channels. Low rank (correlated/LOS): RH can drop to 1 even with many antennas.

Transmit Precoding + Receiver Shaping

Tx Precoding

Transmit x = V·x̃ instead of x̃ directly. Aligns signal with eigen-directions of H.

Rx Shaping

Compute ỹ = Uᴴ·y. Separates the mixed signals into independent streams.

Full Derivation

Start: y = Hx + n  →  substitute x=Vx̃: y = H(Vx̃) + n

Use H=UΣVᴴ: y = UΣVᴴVx̃ + n  →  since VᴴV=I: y = UΣx̃ + n

Multiply by Uᴴ: ỹ = UᴴUΣx̃ + Uᴴn  →  since UᴴU=I:

ỹ = Σ·x̃ + ñ    where ñ = UᴴnΣ is diagonal → ỹᵢ = σᵢ·x̃ᵢ + ñᵢ — each equation is INDEPENDENT!

Noise property: Uᴴ is unitary → it preserves the distribution of n. So n and ñ are identically distributed — the transformation doesn't harm noise statistics.

After SVD:
  x̃₁ ──[σ₁]── ỹ₁   (independent SISO, gain σ₁)
  x̃₂ ──[σ₂]── ỹ₂   (independent SISO, gain σ₂)
  x̃₃ ──[σ₃]── ỹ₃   (independent SISO, gain σ₃)
  No interference between streams!

Benefits: RH× data rate vs SISO; ML decoding complexity becomes LINEAR in RH (not exponential!).

Practice — §10.2
  1. 3×3 full-rank MIMO. How many parallel channels? Multiplexing gain vs SISO?
  2. Why does multiplying noise by Uᴴ not change its statistics? What property ensures this?
  3. A 4×4 channel has rank 2 (antenna correlation). Max multiplexing gain? What does this mean for data rate?

1. R_H=3. Can send 3 independent streams → 3× SISO data rate.

2. Unitary matrix preserves vector length and distribution. If n~CN(0,I), Uᴴn~CN(0,UᴴIU)=CN(0,I) — identical. Key: UᴴU=I preserves covariance.

3. Max gain=2. Only 2 independent streams even with 4 antennas. Data rate=2× SISO. Two degrees of freedom lost — physical spacing and rich scattering are critical.

Section 10.3

MIMO Channel Capacity

What is the absolute maximum data rate over a MIMO channel? Shannon's mutual information gives the answer.

What is Channel Capacity?

Shannon capacity = maximum data rate with arbitrarily small BER. Below capacity → reliable. Above capacity → unavoidable errors.

For static channels with CSIR, capacity = maximum mutual information over all input distributions:

C = max_{p(x)} I(X;Y) = max_{p(x)} [H(Y) − H(Y|X)]Eq. (10.5) · H(Y) = entropy of output · H(Y|X) = H(n) = noise entropy (fixed)

Since H(Y|X) = H(n) is fixed, maximising I(X;Y) ≡ maximising H(Y) — the randomness in the received signal.

Output Covariance Matrix

Ry = H·Rx·Hᴴ + I_{Mr}Eq. (10.6) · Rx = E[xxᴴ] = input covariance · I term = unit noise power

Gaussian Input is Optimal

Key theorem: Among all vectors with given covariance, Gaussian distribution maximises entropy. So optimal input x must be Zero-Mean Circularly Symmetric Complex Gaussian (ZMCSCG). This maximises H(Y) and hence capacity.

Mutual Information Formula

I(X;Y) = B·log₂·det(I_{Mr} + H·Rx·Hᴴ)Eq. (10.7) — First derived by Foschini & Gans and Telatar. det captures all parallel channel gains as a product.

MIMO Capacity Formula

C = max_{Rx: Tr(Rx)=ρ} B·log₂·det(I_{Mr} + H·Rx·Hᴴ)Eq. (10.8) — Optimise over all input covariance matrices satisfying the power constraint

Connection to SVD

Using SVD of H, this simplifies to a sum of SISO capacities:

C ≈ Σᵢ B·log₂(1 + SNRᵢ)MIMO capacity = SUM of individual parallel channel capacities
Practice — §10.3
  1. 2×2 MIMO with H=I₂ (identity), Rx=(ρ/2)I₂, B=1 Hz. Find I(X;Y) using Eq. (10.7). Compare to SISO.
  2. 4 parallel channels with SNRs 20 dB, 15 dB, 10 dB, 0 dB. Find total capacity (B=1 Hz each). Compare to SISO with same total power.

1. HRxHᴴ=(ρ/2)I₂. I+HRxHᴴ=(1+ρ/2)I₂. det=(1+ρ/2)². I(X;Y)=B·log₂((1+ρ/2)²)=2B·log₂(1+ρ/2). Twice the SISO capacity at half per-channel SNR!

2. C=log₂(101)+log₂(32.6)+log₂(11)+log₂(2)=6.66+5.03+3.46+1=16.15 b/s/Hz. SISO with same total power ρ=142.6: C=log₂(143.6)=7.17. MIMO gives 2.25× more capacity!

Section 10.3.1 — CSIT Known

Channel Known at Transmitter: Water-Filling

When the transmitter knows H, it can intelligently distribute power across channels. The optimal strategy has a beautiful physical analogy.

Capacity with CSIT + CSIR

C = max_{ρᵢ: Σρᵢ≤ρ} Σᵢ B·log₂(1 + σᵢ²·ρᵢ)Eq. (10.9) — Sum of parallel SISO capacities with optimised power ρᵢ · R_H = degrees of freedom

Define per-channel SNR at full power: γᵢ = σᵢ²·ρ = σᵢ²·P/σ²

C = max_{Pᵢ: ΣPᵢ≤P} Σ B·log₂(1 + Pᵢ·γᵢ/P)Eq. (10.10) · High SNR: capacity grows linearly with R_H · Low SNR: put all power on strongest channel

Water-Filling Solution

Pᵢ/P = (1/γ₀ − 1/γᵢ) if γᵢ ≥ γ₀, else 0Eq. (10.11) · γ₀ = cutoff "water level" · Channels weaker than γ₀ get ZERO power

Why "Water-Filling"?

Water Level = 1/γ₀
────────────────────────
     ████        ████
     ████  ████  ████
████ ████  ████  ████
 Ch1   Ch2   Ch3   Ch4
(best)(mid)(weak)(dead)
  ↑more  ↑some ↑little ↑none

Base height = 1/γᵢ (inverse of channel quality)
Pour water (total power P) to level 1/γ₀.
Channels with high bases (weak) get no water!
  • Strong channels (large γᵢ) get more power
  • Weak channels get less power
  • Very weak channels (γᵢ < γ₀) get zero power — wasteful to use them
C = Σ_{i: γᵢ≥γ₀} B·log₂(γᵢ/γ₀)Eq. (10.12) — Only active channels contribute
Example 10.2 — Water-Filling (Textbook): ρ=10 dB, B=1 Hz

From Example 10.1: σ₁=1.3333, σ₂=0.5129, σ₃=0.0965. γᵢ = 10·σᵢ²:

Try 3 channels: 3/γ₀ = 1+0.0563+0.380+10.74 = 12.175 → γ₀=0.246. But γ₃=0.093 < 0.246 → Ch3 gets no power!

Redo with 2 channels: 2/γ₀ = 1+0.0563+0.380 = 1.436 → γ₀=1.392. Check: γ₂=2.631 > 1.392 ✓

C = log₂(17.777/1.392) + log₂(2.631/1.392) = 3.674+0.919 = 4.59 bits/s/Hz

Practice — §10.3.1 Water-Filling
  1. 3 parallel channels: γ₁=10, γ₂=4, γ₃=0.5. Total ρ=1. Apply water-filling: find γ₀, power fractions, and capacity (B=1 Hz).
  2. At very high SNR, water-filling allocates nearly equal power. Why does equal power become optimal at high SNR?

1. Try 3: 3/γ₀=1+0.1+0.25+2=3.35→γ₀=0.896. γ₃=0.5<0.896→drop ch3. Try 2: 2/γ₀=1+0.1+0.25=1.35→γ₀=1.481. P₁/P=1/1.481−1/10=0.575. P₂/P=1/1.481−1/4=0.425. C=log₂(10/1.481)+log₂(4/1.481)=2.755+1.432=4.19 b/s/Hz.

2. At high SNR, log₂(1+SNR)≈log₂(SNR). Capacity differences between equal and optimal allocation vanish logarithmically. Jensen's inequality penalty becomes negligible. The marginal value of extra power on a strong channel equals the marginal value on a weaker one when both are already at high SNR.

Section 10.3 — No CSIT

Channel Unknown at Transmitter: Uniform Power

The most common real-world scenario: receiver knows H, transmitter doesn't. Surprisingly, MIMO still delivers remarkable capacity gains.

The Strategy: Equal Power on Every Antenna

⚖️
Intuition: No channel knowledge → no reason to favour any antenna. ZMSW model has no bias. Equal power is optimal for ZMSW channels — proven by Telatar [4].
Rx = (ρ/Mt)·I_{Mt}Equal power allocation input covariance · Each antenna gets ρ/Mt · Signals are uncorrelated
I(x;y) = Σᵢ₌₁^{R_H} B·log₂(1 + γᵢ/Mt)Eq. (10.13) · γᵢ = σᵢ²·ρ · Each channel gets 1/Mt of total power — no per-channel optimisation

Capacity with Outage

Without CSIT, capacity is a random variable — it changes with every channel realisation H. The transmitter doesn't know which rate to use. Solution: pick rate R, accept some probability of failure.

Pout = P(B·log₂·det(I + (ρ/Mt)·HHᴴ) < R)Eq. (10.14) — Probability that channel can't support rate R · Depends on eigenvalue distribution of HHᴴ
Analogy: Driving on an unknown road — you pick a speed (rate) before seeing the conditions (channel). Too fast on a bad road → crash (outage). Outage probability = how often that happens.

The Main Result: Linear Capacity Growth

Using the Law of Large Numbers for random matrices (as Mt → ∞):

(1/Mt)·HHᴴ → I_{Mr}    as Mt → ∞Eq. (10.15) — Large antenna count → channel matrix converges to a scaled identity
C → M·B·log₂(1 + ρ)    where M = min(Mt, Mr)★ MIMO capacity grows LINEARLY with number of antennas — even without CSIT!
★ The Most Important MIMO Result: Even without channel knowledge at the transmitter, MIMO capacity grows linearly with M = min(Mt,Mr). Doubling antennas → doubles capacity. No extra bandwidth, no extra power — just more antennas. This is why MIMO is in every 4G/5G/WiFi system.

Low SNR vs High SNR

High SNR

Capacity limited by degrees of freedom. More antennas → more parallel channels → linear growth: C ∝ M = min(Mt,Mr). Transmit antennas very valuable.

Low SNR

System is collecting energy. Capacity scales only with receive antennas, INDEPENDENT of Mt. Spreading power over Mt antennas or concentrating in 1 gives same result.

Demodulation Without CSIT

Without precoding (x=Vx̃), signals remain interfering at the receiver. ML complexity stays exponential: |X|^Mt. Main practical challenge of MIMO without CSIT — managing decoding complexity through algorithms like sphere decoding or linear detectors (ZF, MMSE).

Non-Ideal Channels

  • Nonzero mean (strong LOS): Uniform power is suboptimal — spatial bias should be exploited
  • Non-identity covariance (correlated antennas): Some directions stronger — equal power wastes resources
  • Dominant direction: Beamforming (Section 10.4) achieves capacity — focuses power like a spotlight
