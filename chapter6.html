Section 6.1

AWGN Channels вҖ” Starting from Zero

Before any formula, we need to understand: what are we sending? What breaks it? And how do we measure performance?

The Signal, Noise, and AWGN

When your phone sends a message, it converts data into a radio wave: s(t). On the way to the receiver, random noise n(t) gets added. So the receiver actually sees:

r(t) = s(t) + n(t)Received = Signal + Noise вҖ” the fundamental wireless equation

AWGN stands for Additive White Gaussian Noise:

WordMeaning
AdditiveNoise adds to the signal
WhiteEqual noise power at all frequencies (like white light)
GaussianNoise values follow a bell curve вҖ” most values near zero, large values rare

The noise n(t) has mean zero and power spectral density (PSD) = NвӮҖ/2.

Signal-to-Noise Ratio (SNR) and Eb/NвӮҖ

рҹ“Ҡ
Why SNR? If signal is strong and noise is small, communication works well. SNR measures that ratio. But SNR alone isn't a fair comparison across different modulations вҖ” Eb/NвӮҖ is.
SNR = Pr / (NвӮҖВ·B)Received power / (noise density Г— bandwidth) В· If bandwidth of complex envelope = B, then noise power = NвӮҖВ·B
Оіb = Eb/NвӮҖ    Оіs = Es/NвӮҖSNR per bit (Оіb) and per symbol (Оіs) вҖ” the universal performance metrics В· For raised cosine pulses (Ts=1/B): SNR = Es/NвӮҖ

Section 6.1.2

Error Probability for BPSK and QPSK

Step-by-step derivation of the most important BER formula in all of wireless communications.

BPSK вҖ” Two Signals, One Bit

BPSK sends bit 0 as sвӮҒ(t) = +Ag(t)cos(2ПҖfct) and bit 1 as sвӮӮ(t) = вҲ’Ag(t)cos(2ПҖfct). Same waveform, phase differs by 180В°.

  вҶҗвҖ” вҲ’вҲҡEb в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ 0 в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ +вҲҡEb вҖ”вҶ’
        bit 1      вҶ‘       bit 0
               decision
               boundary
  dmin = 2A = 2вҲҡEb
Decision: received value > 0 вҶ’ bit 0, < 0 вҶ’ bit 1

Deriving the BER Formula

Step 1 вҖ” From signal space theory: Pb = Q(dmin / вҲҡ(2NвӮҖ))

Step 2 вҖ” Minimum distance: dmin = 2A

Step 3 вҖ” Relate A to Eb: Eb = вҲ«AВІgВІ(t)cosВІ(2ПҖfct)dt = AВІ вҶ’ so A = вҲҡEb

Step 4 вҖ” Substitute:

Pb(BPSK) = Q(вҲҡ(2Eb/NвӮҖ)) = Q(вҲҡ(2Оіb))вҳ… The most important BPSK formula вҖ” memorise this
Example вҖ” BPSK at Оіb = 10 dB

Оіb = 10^(10/10) = 10. BER = Q(вҲҡ20) = Q(4.47) вүҲ 3.9Г—10вҒ»вҒ¶. About 4 bits per million are wrong вҖ” fine for voice!

QPSK вҖ” Two Bits Per Symbol

QPSK splits the signal into I (in-phase) and Q (quadrature) branches. Each branch is independently BPSK. With perfect phase recovery, the branches are orthogonal вҶ’ each has the same BER as BPSK.

Pb(QPSK) = Q(вҲҡ(2Оіb))вҳ… Same as BPSK вҖ” even though QPSK sends 2Г— the data! Es = 2Eb вҶ’ Оіs = 2Оіb
рҹӨҜ
Why same BER? QPSK energy splits between two orthogonal dimensions. Each dimension behaves exactly like BPSK with the same Eb. You get 2Г— data rate for FREE in terms of BER.
Ps(QPSK) = 1 вҲ’ (1 вҲ’ Q(вҲҡ(2Оіb)))ВІ = 1 вҲ’ (1 вҲ’ Q(вҲҡОіs))ВІSymbol error: error if either I or Q branch makes an error

Nearest neighbor approximation: Ps вүҲ 2Q(вҲҡОіs). With Gray coding: Pb вүҲ Ps/2.

Example 6.1 вҖ” QPSK at Оіb = 7 dB (Textbook)

Оіb = 10^(0.7) = 5.012. Pb = Q(вҲҡ10.024) = 7.726Г—10вҒ»вҒҙ. Ps = 1вҲ’(1вҲ’7.726Г—10вҒ»вҒҙ)ВІ = 1.545Г—10вҒ»Ві. Gray approx: Pb вүҲ Ps/2 = 7.723Г—10вҒ»вҒҙ вң“. Nearest neighbour (Оіs=10.024): Ps вүҲ 2Q(вҲҡ10.024) = 1.545Г—10вҒ»Ві вң“

Section 6.1.3

Error Probability for MPSK

M symbols equally spaced on a circle. As M increases, symbols crowd together вҖ” and errors increase.

M-PSK Constellation

Symbols: si1 = AВ·cos(2ПҖ(iвҲ’1)/M), si2 = AВ·sin(2ПҖ(iвҲ’1)/M). All on a circle, radius A. Symbol energy Es = AВІ, so Оіs = AВІ/NвӮҖ.

8-PSK (M=8, 3 bits/symbol):
         вҖў
    вҖў         вҖў
  вҖў               вҖў    All 8 points on circle,
    вҖў         вҖў        spaced 360В°/8 = 45В° apart.
         вҖў

An error occurs when noise rotates the received signal outside its angular decision slice (ВұПҖ/M). For M>4, the exact Ps integral has no closed form вҖ” we use approximations.

Minimum Distance and Nearest Neighbour Approx

dmin = 2AВ·sin(ПҖ/M)Distance between adjacent symbols on the circle В· As MвҶ‘, sin(ПҖ/M)вҶ“ вҶ’ symbols get closer вҶ’ more errors
Ps вүҲ 2Q(вҲҡ(2Оіs)В·sin(ПҖ/M))вҳ… Key MPSK formula В· With Gray coding: Pb вүҲ Ps/logвӮӮM
Example 6.2 вҖ” 8-PSK vs 16-PSK at Оіb = 15 dB (Textbook)

Оіb = 31.62. 8-PSK: Оіs = 3Г—31.62 = 94.87. Ps вүҲ 2Q(вҲҡ189.74 Г— sin(ПҖ/8)) = 1.355Г—10вҒ»вҒ·. Pb = Ps/3 = 4.52Г—10вҒ»вҒё.

16-PSK: Оіs = 4Г—31.62 = 126.49. Ps вүҲ 2Q(вҲҡ252.98 Г— sin(ПҖ/16)) = 1.916Г—10вҒ»Ві. Pb = Ps/4 = 4.79Г—10вҒ»вҒҙ.

16-PSK Pb is ~10,000Г— worse than 8-PSK! More symbols on the same circle вҶ’ closer spacing вҶ’ noise causes more errors.

Section 6.1.4

Error Probability for MPAM and MQAM

QAM uses both amplitude AND phase вҖ” giving much more efficient use of signal space than PSK.

MPAM вҖ” Amplitude Levels

Constellation: Aбөў = (2iвҲ’1вҲ’M)d, i=1..M. Inner points have 2 neighbours; outer points have 1. Average symbol energy: Д’s = (MВІвҲ’1)dВІ/3.

Ps = 2(MвҲ’1)/M В· Q(вҲҡ(6ОіМ„s/(MВІвҲ’1)))Equation (6.21) вҖ” MPAM symbol error in terms of average SNR

MQAM вҖ” Amplitude + Phase

Square M-QAM (M=LВІ) = two independent L-PAM systems on I and Q axes. Each axis gets half the energy.

Ps,branch = 2(вҲҡMвҲ’1)/вҲҡM В· Q(вҲҡ(3ОіМ„s/(MвҲ’1)))Equation (6.22) вҖ” per-branch error (note factor 3, not 6)
Ps = 1 вҲ’ (1 вҲ’ Ps,branch)ВІEquation (6.23) вҖ” exact MQAM symbol error
Ps вүҲ 4Q(вҲҡ(3ОіМ„s/(MвҲ’1)))Equation (6.24) вҖ” nearest-neighbour approximation (4 neighbours for inner points)
Example 6.3 вҖ” 16-QAM vs 16-PSK at Оіb = 15 dB (Textbook)

Оіs = 4Г—31.62 = 126.49. 16-QAM exact: Ps = 7.37Г—10вҒ»вҒ·. Nearest-neighbour: Ps вүҲ 9.82Г—10вҒ»вҒ·.

16-PSK (from Ex. 6.2): Ps = 1.916Г—10вҒ»Ві вҖ” roughly 4 orders of magnitude worse!

Why QAM wins: PSK uses only phase (1D circle). QAM uses amplitude + phase (2D grid). Same energy вҶ’ larger minimum distance вҶ’ fewer errors.

Practical note: MQAM needs both phase and amplitude estimates at the receiver. Channel gain is obtained via pilot symbols; amplitude scaling is called AGC (Automatic Gain Control). Imperfect estimates cause extra errors.

Section 6.1.5

Error Probability for FSK and CPFSK

FSK changes frequency instead of phase or amplitude. Simpler receiver, but less power-efficient than PSK.

Binary FSK

Bit 0 вҶ’ frequency fвӮҒ, bit 1 вҶ’ frequency fвӮӮ. Signal: sбөў(t) = AвҲҡ(2/Tb)В·cos(2ПҖfбөўt). Eb = AВІ, so Оіb = AВІ/NвӮҖ.

Coherent detector computes z = sвӮҒ+nвӮҒ вҲ’ sвӮӮвҲ’nвӮӮ. Error when noise overcomes signal. Since nвӮҒвҲ’nвӮӮ has variance NвӮҖ (sum of two NвӮҖ/2 variances):

Pb(FSK) = Q(вҲҡОіb)Equation (6.31) вҖ” Compare: BPSK has Q(вҲҡ(2Оіb)) вҶ’ BPSK is 3 dB better than FSK

For noncoherent MFSK (receiver doesn't know phase): Ps = ОЈвӮҳ(вҲ’1)^(m+1)В·C(MвҲ’1,m)В·(1/(m+1))В·exp(вҲ’mОіs/(m+1))

CPFSK

Regular FSK has abrupt phase jumps at symbol boundaries. CPFSK makes frequency change smoothly вҖ” phase is continuous. Used in Bluetooth (GFSK), GSM (GMSK). CPFSK has memory (current phase depends on all previous phases), making error analysis complex вҖ” must work over multiple symbols, not just one.

Section 6.2

Alternate Q-Function Representation

Craig's 1991 discovery: rewrite Q(z) with a finite integral вҖ” making BER averaging over fading channels analytically tractable.

Why Standard Q(z) Is Awkward

Q(z) = вҲ«_z^вҲһ (1/вҲҡ(2ПҖ))В·e^(вҲ’xВІ/2) dxLower limit depends on z В· upper limit is вҲһ В· no closed-form solution for most operations

Especially difficult when we need E[Q(вҲҡ(2Оі))] over random Оі in fading channels.

Craig's Alternate Form (1991)

Q(z) = (1/ПҖ) вҲ«вӮҖ^(ПҖ/2) exp(вҲ’zВІ/(2sinВІПҶ)) dПҶ,   z > 0Eq. (6.43) вҖ” Fixed limits [0, ПҖ/2] В· independent of z В· Gaussian in z вҶ’ easy to average!

Application to BPSK: Pb = (1/ПҖ)вҲ«вӮҖ^(ПҖ/2) exp(вҲ’Оіb/sinВІПҶ) dПҶ (Eq. 6.44)

Exact MPSK error (no approximation!): Ps = (1/ПҖ)вҲ«вӮҖ^((MвҲ’1)ПҖ/M) exp(вҲ’gВ·Оіs/sinВІПҶ) dПҶ where g = sinВІ(ПҖ/M) (Eq. 6.45)

Why it matters for fading: Using Craig's form, E[Q(вҲҡ(2Оі))] = (1/ПҖ)вҲ«вӮҖ^(ПҖ/2) E[exp(вҲ’Оі/sinВІПҶ)] dПҶ. The inner expectation is the MGF of Оі вҖ” which has simple closed forms for Rayleigh, Rician, and Nakagami distributions. This unlocks closed-form average BER for all common fading models.
Section 6.3

Fading вҖ” When SNR Goes Random

In real wireless, signal strength fluctuates randomly. This section shows how that changes performance вҖ” often catastrophically.

What is Fading?

In AWGN, Оіs is constant. In wireless, received power varies due to shadowing (large obstacles) and multipath fading (multiple reflected paths add constructively or destructively). Оіs becomes a random variable with distribution pОіs(Оі).

рҹҸҷпёҸ Urban canyon: Walking between skyscrapers, your call arrives via direct path AND building reflections. Sometimes they add (clear call), sometimes cancel (call drops). Moving a few centimeters can change signal strength dramatically!

Rayleigh Fading SNR Distribution

In Rayleigh fading (no line-of-sight), signal amplitude r is Rayleigh distributed. After change of variables, the instantaneous SNR Оі follows an exponential distribution:

Section 6.3.1

Outage Probability

When the channel fades so deeply the link becomes completely unusable вҖ” that's an outage. The outage probability directly measures system reliability.

Pout = P(Оі < ОівӮҖ) = 1 вҲ’ e^(вҲ’ОівӮҖ/ОіМ„s)Eq. (6.46, 6.47) вҖ” Rayleigh fading outage probability В· ОівӮҖ = minimum SNR for acceptable performance
ОіМ„s = вҲ’ОівӮҖ / ln(1 вҲ’ Pout)Eq. (6.48) вҖ” Required average SNR for target outage
Fd = вҲ’10В·log[вҲ’ln(1 вҲ’ Pout)]Fade margin in dB вҖ” extra SNR needed above ОівӮҖ to guarantee 100В·(1вҲ’Pout)% reliability
Example 6.4 вҖ” Required SNR for BPSK, 95% Coverage (Textbook)

Target: Pb < 10вҒ»вҒҙ for 95% of locations. BPSK in AWGN at 10вҒ»вҒҙ needs ОівӮҖ = 8.5 dB.

Pout = 0.05: ОіМ„b = вҲ’10^0.85 / ln(0.95) = 10.85/0.0513 = 21.4 dB.

8.5 dB needed in AWGN, 21.4 dB needed in Rayleigh fading вҖ” 12.9 dB penalty just to combat fading!

Set A вҖ” AWGN
  1. Compare required Eb/NвӮҖ (dB) for BPSK, 8-PSK, and 16-QAM to achieve Pb вүӨ 10вҒ»вҒө in AWGN. Q(4.42) вүҲ 10вҒ»вҒө.
  2. For 64-QAM at Оіb = 25 dB: find Оіs, Ps (nearest-neighbour), and Pb (Gray coding).

Set B вҖ” Fading and Design
  1. A mobile user is in Rayleigh fading with ОіМ„b = 18 dB. Does BPSK achieve Pb вүӨ 10вҒ»Ві? If not, what ОіМ„b is needed?
  2. A 5G cell needs ОівӮҖ = 12 dB and Pout = 2%. Find required ОіМ„ in dB.
  3. Compare power penalty of Rayleigh fading vs AWGN for BPSK at Pb = 10вҒ»вҒҙ.

1. ОіМ„b=63.1. PМ„bвүҲ1/(4Г—63.1)=3.96Г—10вҒ»Ві > 10вҒ»Ві. Fails! Need: 1/(4ОіМ„b)=10вҒ»Ві вҶ’ ОіМ„b=250 вҶ’ 24 dB.

2. ОівӮҖ=15.85. ОіМ„=вҲ’15.85/ln(0.98)=вҲ’15.85/(вҲ’0.0202)=784.6 вҶ’ 28.9 dB.

3. AWGN: Q(вҲҡ(2Оіb))=10вҒ»вҒҙ вҶ’ ОіbвүҲ6.9 вҶ’ 8.4 dB. Fading: 1/(4ОіМ„b)=10вҒ»вҒҙ вҶ’ ОіМ„b=2500 вҶ’ 34 dB. Penalty = 34вҲ’8.4 = 25.6 dB вҖ” over 350Г— more power!
